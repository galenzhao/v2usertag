Sync your custom user tags across devices using GitHub.

GitHub Connection

We need a Fine-Grained Personal Access Token (PAT) with repo permissions to sync your tags.

How to create a Token?
  1. Go to GitHub Tokens (beta).
  2. Click Generate new token.
  3. Under Repository access, choose "Only select repositories" and pick your target repo.
  4. Under Permissions -> Repository permissions, find Contents and set it to Read and write.
  5. Scroll down and click Generate token.
  6. Copy the token (starts with github_pat_) and paste it below.
